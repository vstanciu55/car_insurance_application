Welcome to the Car Insurance Web Application Assignment

Steps needed to start the project

1. Open Command Line
2. Run 'bundle install' (Make sure Ruby on Rails is installed)
3. Run 'rails/rake db:migrate'
4. Run 'rails s' to start the server
5. Open browser at the localhost

The application gives text information on how to proceed.

If needed, change the Domain found in the Rest-Client.rb if it is run from a different platform
